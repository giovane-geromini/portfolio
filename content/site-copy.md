## PROJETOS

### PlantaCheck — Controle inteligente de cuidados com plantas
Status: MVP funcional em desenvolvimento contínuo

**Problema**
Muitas pessoas têm plantas em casa, mas acabam esquecendo de regar, não sabem se estão cuidando corretamente ou não possuem histórico confiável de cuidados.

**Solução**
Aplicativo web que permite organizar plantas por casa e ambientes, registrar eventos de rega e exposição ao sol, e calcular automaticamente o status de cada planta com base em regras definidas pelo usuário.

**Funcionalidades**
- Autenticação de usuários
- Casas compartilhadas
- Cadastro de plantas
- Registro de rega e sol (inclusive em lote)
- Cálculo automático de status
- Dashboard central
- Deploy em produção

**Principais desafios técnicos**
- Modelagem de dados orientada a eventos
- Separação entre estado atual e histórico
- Definição de regras de negócio
- Evolução de banco sem quebra
- UI simples para uso diário

**Stack**
- Next.js
- TypeScript
- Supabase
- Tailwind CSS
- Vercel
